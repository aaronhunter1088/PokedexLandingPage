About this Pokédex Application This Pokédex application was built originally using Spring Boot as a fun project in order to practice moreso with the Spring framework. Later on, I needed to learn Angular, so I decided to clone the Pokédex application since I already had a working version. This allowed me to focus on learning Angular semantics and the best practices without worries about what I was building. The Spring Boot application is 100% Java and Spring Boot. The Angular app is a 100% JavaScript/Angular application.
One day, I decided to create a version using both technologies together. Using Angular as my font-end and Spring Boot as my back-end, the third version of this Pokédex application was born.
This landing page was created as a means to present all three versions in one place. This app works with the PokéAPI to fetch and display various Pokémon data.
